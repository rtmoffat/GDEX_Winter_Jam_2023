# GDEX_Winter_Jam_2023
GDEX Winter Jam 2023

To use locally:

Clone the repo:

git clone https://github.com/rtmoffat/GDEX_Winter_Jam_2023.git

Pull down all the branches:

git pull --all

switch to the dev branch

git checkout dev

Make your changes

Stage your changes

git add .

commit your changes

git commit -m "I changed these things"

push your changes

git push
